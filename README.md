### Supply Chain Optimization Engine
![Stack](https://img.shields.io/badge/FastAPI_%7C_RandomForest_%7C_PuLP_%7C_OR_Tools-0f172a?style=flat-square)

> 3-stage ML + Operations Research pipeline for pharmaceutical network optimization across 352 nodes.

| Objective | Baseline Approach | Optimized Solution | Improvement |
| :--- | :--- | :--- | :--- |
| **Demand Forecasting** | — | 600-tree Random Forest (13 temporal features) | 6.88% MAPE, R² = 0.9919 |
| **Routing Optimization** | Nearest-Neighbor Heuristic | CVRP via Google OR-Tools | ▼ 56.79% cost reduction |
| **Resource Allocation** | Greedy Heuristic | Global LP Optimization via PuLP | ▼ 64.95% cost reduction |
