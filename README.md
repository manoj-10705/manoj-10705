### 🚚 Intelligent Warehouse-to-Store Supply Chain Optimizer
**Tech Stack:** FastAPI | Random Forest | PuLP | OR-Tools | React

ML-driven optimization pipeline for pharmaceutical distribution across 352-node network.

- **Forecasting Accuracy:** 6.88% MAPE with R² = 0.9919 using 600-tree Random Forest
- **Cost Reduction:** 56.79% routing cost reduction via CVRP optimization
- **Allocation Optimization:** 64.95% cost reduction through PuLP global optimization
- Multi-parameter disruption simulation with formal feasibility boundary analysis

**Technical Highlights:**
- 3-stage ML + OR pipeline (Random Forest → LP allocation → CVRP routing)
- 13 temporal features for demand forecasting
- Resilience quantification under demand/capacity/route perturbations
